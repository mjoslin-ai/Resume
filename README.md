# Resume

Easily tailor resume for different companies and roles.

## Getting Started

### Dependencies

* Windows 11
* TeX Live
* VS Code
* LaTeX Workshop from VS Code Marketplace
* Remote Development from VS Code Marketplace

### Generating Resume

* Edit data.json
* Run json_to_tex.py
